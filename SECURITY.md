# Security Policy

## Supported Versions

Only the latest released version is supported with security updates.

## Reporting a Vulnerability

Please do not disclose security issues publicly before they are addressed.

Report vulnerabilities privately to the project maintainer with:

- A description of the issue
- Steps to reproduce
- Potential impact
- Suggested mitigation (if known)

The project is designed to:
- Process documents in memory whenever practical.
- Avoid logging document contents or field values.
- Require authenticated API access.
